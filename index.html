RSDK by Canownueasy

RuneScape Development Kit

RSDK is an effort to provide a reliable and strong framework with easy-to-implement systems which are of ultimate extension capabilities.

Dependencies

jBoss Netty

Install

Run in JVM (1.6+) with Eclipse
